# 🎬 FluxReviews

FluxReviews is a modern, high-fidelity movie review platform featuring real-time data synchronization, TMDB-powered movie details auto-fill, and glassmorphic aesthetics.

## 🚀 Key Features & Visual Upgrades

* **Real-time Sync**: Direct integration with Firebase Database for instant catalog updates without page refreshes.
* **TMDB Auto-Fill**: Query TMDB to automatically retrieve posters, plot summaries, release years, and genres.
* **Cinematic Backdrop Blurs**: Movie detail modal dynamically renders a blurred, shaded poster image backdrop (`blur(60px)`) for a high-end streaming feel.
* **Pulsing Loading Skeletons**: Custom shimmering card placeholders render during loading states to prevent blank screens on slow network connections.
* **Decoupled Dual-Column Scrolling**: The admin page features independent scrolling columns, letting users fill editing forms while keeping catalog grids stationary.
* **Sharing & Likes**: User engagement via a local-storage heart toggle and instant clipboard link sharing.
